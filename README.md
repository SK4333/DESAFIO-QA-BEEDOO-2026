DESAFIO QA – BEEDOO 2026
1. Objetivo

O objetivo desta aplicação é permitir que usuários cadastrem e visualizem cursos.
O sistema funciona como um gerenciador de cursos, permitindo adicionar novos cursos e listar os cursos cadastrados.

2. Escopo dos Testes

Durante os testes foram avaliadas as seguintes funcionalidades:

Cadastro de cursos

Listagem de cursos

Validação de campos obrigatórios

Comportamento da aplicação em cenários inválidos

3. Estratégia de Teste

Os testes foram realizados com foco em testes funcionais, verificando se as funcionalidades da aplicação atendem aos requisitos esperados.

Tipos de testes aplicados:

Testes de cadastro de cursos

Testes de validação de campos obrigatórios

Testes de exibição de dados na listagem

Testes com dados válidos e inválidos
