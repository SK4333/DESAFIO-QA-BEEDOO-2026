# DESAFIO QA BEEDOO 2026

## Objetivo da aplicação

A aplicação permite cadastrar e visualizar cursos. 
O objetivo do sistema é gerenciar cursos cadastrados pelos usuários.

## Principais fluxos

Cadastro de cursos  
Listagem de cursos  

## Pontos críticos para teste

Validação de campos obrigatórios  
Cadastro de cursos  
Exibição correta dos cursos na lista
